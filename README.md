# 闲置集市 🛍️

一个功能完整的二手物品交易平台，基于 Node.js + Express + SQLite 构建。

## 功能特性

- 👤 用户注册 / 登录 / 个人中心
- 📷 商品发布（支持多图上传）
- 🔍 商品搜索、分类筛选、多种排序
- ❤️ 收藏商品
- 💬 站内消息（联系卖家）
- 📦 商品状态管理（在售/预订/已售）

## 技术栈

- **后端**: Node.js + Express
- **数据库**: SQLite3
- **前端**: 原生 HTML/CSS/JavaScript

## 本地运行

```bash
npm install
npm start
```

访问 http://localhost:3000
